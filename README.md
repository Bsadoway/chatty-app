Chatty App
=====================

A simple chat app built with React and Websockets.

### Usage

Chatty-App:

Clone the repo and install dependencies
```
npm install
npm start
open http://localhost:3000
```

Chatty-App Server:
```
oppen the chatty-server/ folder
npm install
npm start
```
Both must be running for the app to work.
